# CatchCam

## Описание

Интерактивное приложение с использованием компьютерного зрения. Управление осуществляется с помощью жестов рук через веб-камеру. Проект включает в себя несколько мини-игр и режимов взаимодействия.

## Функционал

- **Управление жестами**: Используйте движения рук и пальцев для взаимодействия с виртуальными объектами.
- **Игры**:
  - **2048**: Классическая головоломка с управлением жестами.
  - **Touch Cubes**: Игра на реакцию и точность.
  - **Collect All Cubes**: Режим сбора объектов.
- **Дополнительные возможности**:
  - Детекция лица (Face Mesh)
  - Удаление/замена фона
  - Запись видео игрового процесса
  - Отображение FPS

## Требования

- Python 3.x
- Веб-камера

## Установка

1. Скачайте проект.
2. Установите необходимые библиотеки, используя файл `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```
   _Зависимости включают: opencv-python, mediapipe, cvzone, numpy._

## Использование

Для запуска приложения выполните следующую команду в терминале:

```bash
python main.py
```

Для создания .exe файла используйте команду:
```bash
pyinstaller --onefile --collect-all mediapipe main.py
```

### Управление в приложении

| Клавиша      | Действие                                |
| ------------ | --------------------------------------- |
| **ESC**      | Выход из приложения                     |
| **F**        | Включить / Выключить запись видео       |
| **Ctrl + Z** | Вернуться в главное меню                |
| **Tab**      | Вывести отладочную информацию в консоль |
